**Requeriments Engeneering**

• Non-Functional Requeriments:

→ Text-based Java system;

→ Data persistence with JDBC;

→ Project build with Maven;

→ Factory and DAO design patterns;

• Functional Requeriments: 

1) Freight:
• Operations: change payment status, find freights not paid, find freights paid, list all freights, add new freight, update a freight, delete a freight.

2) Driver:
• Operations: list all Drivers, find an Driver, update a Driver, add new Driver and delete a Driver.

3) Vehicle
• Operations: add new Vehicle, find a Vehicle, update a Vehicle, delete a Vehicle, list all Vehicles.
